To run this web application, do the following:

1) Install Vagrant (https://www.vagrantup.com/downloads.html)
2) Open a command-line window and navigate to the vagrant directory
3) Type vagrant up into the command line to start a new vagrant server
4) When the server has finished setting itself up, type vagrant ssh to log into the server
5) After logging it, vagrant will prompt you to enter cd /vagrant to go to your virtual directory. Type cd /vagrant and go to the directory.
6) navigate to the catalog directory with cd catalog.
7) Run the server script with python project.py.
8) In a web browser window, navigate to http://localhost:5000 to run the web app.
