README

PROJECT: Catalog Project


ABOUT:
The files within the chisel_catalog folder and create the database, tables, and function calls necessary to facilitate a web app allowing users to login with OAuth accounts via Facebook and Google.


INSTRUCTIONS:
1. Install Vagrant
2. Boot up Vagrant with terminal command “vagrant up”
3. Navigate to the chisel_catalog project folder
4. Run the setup_chisel_db.py file through the terminal to create the database and tables
	> python setup_chisel_db.py
5. Run the chisel_app.py file through the terminal to test the function calls in the tournament.py file
	> python chise_app.py
